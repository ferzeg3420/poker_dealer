Hand Rankings


Royal Flush (Best) Five cards of the same suit: must go from 10 through A.

Straight Flush Any five cards of the same suit in sequence. The sequence does not wrap around, so K, A, 2, 3, 4 is not a valid sequence.

Four of a Kind Four cards with the same value. The fifth card can be anything.

Full House One set of three cards of the same value and another set of two other cards of the same value (but different from the first set).

Flush Any five cards of the same suit.

Straight Any sequence of values (does not wrap around so K, A, 2 and the like are invalid).

Three of a Kind Three cards of the same value. The last two can be anything.

Two Pair One set of two cards that have the same value and another set of two cards of the same value (but different value from the first set).

Pair Two cards of the same value and the rest can be anything.

High Card (Worst) Any five cards not in sequence and that don't have all the same suit.
